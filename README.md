# MagnifyingLayout
## 自定义Layout 实现布局跟随手指移动放大局部布局效果、布局内支持任意View
使用步骤:

**Step1:** 
项目根目录build文件中配置jitpack maven
```
 repositories {
        maven { url 'https://jitpack.io' }
    }
```

**Step2:**
app目录build文件中配置引用
```
    implementation 'com.github.ljlstudio:MagnifyingLayout:Tag' ..tag为最新release 版本（1.0.1）
```
